My Projects

Smart Traffic Automation System

Empowering urban transportation, the Smart Traffic Automation System utilizes OpenCV and deep learning algorithms to strategically regulate traffic dynamics. Through real-time video analysis, it optimizes signal sequencing, identifies congestion patterns, and enhances road safety, contributing to the advancement of efficient and secure urban mobility solutions. Additionally, it provides trends in traffic on a hourly, daily, weekly as well as a monthly basis to perform analytics. Emergency vehicles are given priority.

Predictive Modelling of Refinery Chemical Demand

LSTM & ANN based model was deployed to map consumption of a specific chemical inside refinery and model was used to define weekly demand of chemical basis which external procurement was planned and schedule. Model is currently being used for essential chemical procurement within refinery.

CollabMP3

CollabMP3, driven by K-Nearest Neighbors (KNN) algorithm. Leveraging user preferences and real-time data, it intelligently suggests live music events tailored to individual tastes. Enhancing music discovery experiences, it redefines personalized event recommendations for music enthusiasts. It can connect to the user’s spotify account and recommend tracks based on the user’s real time music track.

Anamoly Detection in Traffic Network

This project implements unsupervised learning techniques for anomaly detection in network traffic data. It utilizes two powerful methods: Isolation Forests and Autoencoders. These techniques aim to identify unusual patterns potentially indicative of security breaches or system malfunctions.

Face Detection using Flask

Flask-based web application that performs real-time face detection using computer vision. The app processes images uploaded by users, detects faces, and displays results, providing a simple interface for integrating facial recognition into web services.

ToDo using NestJs

A task management app using NestJS that enables users to add, update, and delete tasks. The application features RESTful APIs for CRUD operations, user authentication, and task categorization, offering a robust and scalable solution for managing daily to-dos.
